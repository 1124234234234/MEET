# 会议智能分析系统 - 安装和使用说明

## 一、系统要求

- **操作系统**: Windows 10/11
- **Python**: 3.10 或更高版本
- **FFmpeg**: 用于音频处理（可选但推荐）
- **内存**: 建议 8GB 或以上

---

## 二、安装步骤

### 方法一：使用本地启动脚本（推荐）

1. **双击运行 `start-local.bat`**
   - 脚本会自动：
     - 检查并创建 Python 虚拟环境
     - 安装所有依赖包
     - 检查 FFmpeg
     - 启动系统

2. **访问系统**
   - 打开浏览器，访问：http://localhost:5000

---

### 方法二：手动安装

#### 1. 安装 Python

1. 访问 https://www.python.org/downloads/
2. 下载 Python 3.10 或更高版本
3. 运行安装程序，**务必勾选 "Add Python to PATH"**
4. 验证安装：打开命令行，运行 `python --version`

#### 2. 安装依赖

```bash
# 进入项目目录
cd d:\games\新建文件夹 (7)\voice-reco

# 创建虚拟环境（可选但推荐）
python -m venv venv
venv\Scripts\activate

# 安装依赖包
pip install -r requirements.txt
```

#### 3. 安装 FFmpeg（可选但推荐）

1. 访问 https://ffmpeg.org/download.html
2. 下载 Windows 版本（推荐使用预编译的二进制文件）
3. 解压到某个目录
4. 将 ffmpeg 的 `bin` 目录添加到系统环境变量 `PATH`

#### 4. 启动系统

```bash
python app.py
```

然后在浏览器访问 http://localhost:5000

---

## 三、项目备份

### 方法一：使用备份脚本（最简单）

1. 双击运行 `backup.bat`
2. 备份文件会自动保存到 `backup/` 目录，文件名包含时间戳

### 方法二：使用 Git 备份到 GitHub

#### 1. 安装 Git

1. 访问 https://git-scm.com/download/win
2. 下载并安装 Git（使用默认设置即可）

#### 2. 初始化 Git 仓库

```bash
cd d:\games\新建文件夹 (7)\voice-reco
git init
git add .
git commit -m "Initial commit"
```

#### 3. 推送到 GitHub

1. 在 https://github.com/new 创建一个新仓库
2. 按 GitHub 页面提示的步骤操作：

```bash
# 将仓库关联到 GitHub
git remote add origin https://github.com/你的用户名/你的仓库名.git
git branch -M main
git push -u origin main
```

#### 4. 后续提交更改

```bash
git add .
git commit -m "描述你的更改"
git push
```

## 四、功能介绍

### 1. 实时录音与转写
- 点击「开始录音」按钮
- 系统会实时转写并显示说话人

### 2. 上传音频文件
- 支持 MP3、WAV、M4A 等格式
- 上传后自动进行分析

### 3. 知识库管理
- 管理政策文件、风险关键词、必传要点
- 支持添加、编辑、删除、搜索

### 4. 合规检查
- 自动比对会议内容与知识库
- 生成合规评分报告

### 5. 会议记录
- 查看历史会议记录
- 导出报告（PDF/Word）

## 五、常见问题

### Q: 依赖包安装失败？
A: 尝试更换镜像源：
```bash
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### Q: 录音功能不工作？
A: 检查浏览器是否允许麦克风权限

### Q: 如何停止系统？
A: 在命令行窗口按 `Ctrl + C`
