一、需求分析

1.1 业务背景

本系统是基于语音识别技术的会议智能分析平台，核心目标是实现会议内容的自动转写、分析和合规校验，提升会议管理效率和质量把控能力。

1.2 功能需求

根据需求文档，系统主要包含两大核心功能模块：

1.2.1 音频分析功能

- 实时语音转写：将会议全程语音实时转写为文字
- 说话人分离：支持多人同时发言、交替发言场景，区分不同人员发言内容
- 关键词提取：自动抓取会议中的核心关键词和高频讨论内容
- 主题分析：对全文内容进行语义分析，识别讨论主题和核心方向
- 音频预处理：自动完成降噪、回声消除和语音增强

1.2.2 合规比对功能

- 知识库管理：后台支持管理人员维护会议文件知识库
- 文件录入：可录入、修改、更新各类会议文件、会议精神、强制传达要点及管控关键词
- 动态维护：根据政策迭代情况动态维护知识库内容
- 语义相似度比对：实时将语音转写内容与政策知识库进行语义相似度比对
- 要点核查：逐条核查既定必传要点，自动识别未覆盖、未提及的内容
- 合规评分：自动生成0-100分合规评分，评分权重支持自定义配置
- 风险识别：识别偏离主题、表述不当、消极负面等风险内容

1.3 非功能需求

- 准确性：语音识别准确率≥95%，语义分析准确率≥90%
- 实时性：语音转写延迟≤3秒，合规比对延迟≤5秒
- 可靠性：系统可用性≥99.5%
- 扩展性：支持至少100人并发会议分析

四、功能模块详细设计

4.1 音频预处理模块

4.1.1 功能说明

对输入音频进行预处理，提升语音识别准确率。

4.1.2 处理流程

1. 音频格式转换：统一转换为WAV格式，采样率16kHz，单声道
2. 噪声检测：使用统计方法检测背景噪声
3. 降噪处理：基于频谱减法进行噪声抑制
4. 回声消除：使用自适应滤波器消除空间回声
5. 语音增强：基于谱增强技术提升语音清晰度

4.1.3 技术实现

```python
import librosa
import numpy as np
from scipy import signal

def preprocess_audio(audio_path):
    # 加载音频
    y, sr = librosa.load(audio_path, sr=16000, mono=True)
    
    # 噪声检测与降噪
    noise_level = np.mean(np.abs(y[:int(sr*0.5)]))
    y_denoised = y - noise_level * 0.3
    
    # 回声消除（简化实现）
    b, a = signal.butter(4, [300, 3000], btype='band', fs=sr)
    y_filtered = signal.filtfilt(b, a, y_denoised)
    
    # 语音增强
    y_enhanced = librosa.effects.preemphasis(y_filtered, coef=0.97)
    
    return y_enhanced, sr
```

```python
from pyannote.audio import Pipeline

def speaker_diarization(audio_path, hf_token):
    pipeline = Pipeline.from_pretrained(
        "pyannote/speaker-diarization-3.1",
        use_auth_token=hf_token
    )
    
    diarization = pipeline(audio_path)
    
    speaker_segments = []
    for segment, _, speaker in diarization.itertracks(yield_label=True):
        speaker_segments.append({
            "start": segment.start,
            "end": segment.end,
            "speaker": speaker
        })
    
    return speaker_segments
```

4.4 关键词提取模块

4.4.1 功能说明

4.4.2 技术实现

```python
from rake_nltk import Rake
import jieba

def extract_keywords(text, language="zh", top_n=10):
    if language == "zh":
        # 使用jieba分词
        words = jieba.lcut(text)
        # 过滤停用词
        stopwords = set(["的", "是", "在", "和", "了", "有", "我", "他", "她", "它"])
        filtered_words = [w for w in words if w not in stopwords and len(w) > 1]
        # 统计词频
        word_freq = {}
        for word in filtered_words:
            word_freq[word] = word_freq.get(word, 0) + 1
        # 返回高频词
        return sorted(word_freq.items(), key=lambda x: x[1], reverse=True)[:top_n]
    else:
        r = Rake()
        r.extract_keywords_from_text(text)
        return r.get_ranked_phrases()[:top_n]
```

4.7 合规比对模块

4.7.1 功能说明

将语音转写内容与政策知识库进行语义相似度比对，生成合规评分。

4.7.2 合规评分算法

```python
def calculate_compliance_score(transcription, knowledge_base_items):
    """
    计算合规评分（0-100分）
    
    评分权重：
    - 语义相似度匹配：40%
    - 必传要点覆盖率：30%
    - 风险内容检测：20%
    - 关键词命中：10%
    """
    
    score_components = {
        "semantic_similarity": 0,
        "point_coverage": 0,
        "risk_detection": 0,
        "keyword_matching": 0
    }
    
    # 1. 语义相似度匹配（40分）
    total_similarity = 0
    item_count = len(knowledge_base_items)
    for item in knowledge_base_items:
        similarity = calculate_semantic_similarity(transcription, item.content)
        total_similarity += similarity
    score_components["semantic_similarity"] = (total_similarity / max(item_count, 1)) * 40
    
    # 2. 必传要点覆盖率（30分）
    all_required_points = []
    covered_points = []
    for item in knowledge_base_items:
        all_required_points.extend(item.required_points)
    
    for point in all_required_points:
        if point.lower() in transcription.lower():
            covered_points.append(point)
    
    coverage_rate = len(covered_points) / max(len(all_required_points), 1)
    score_components["point_coverage"] = coverage_rate * 30
    
    # 3. 风险内容检测（20分）
    risk_keywords = []
    for item in knowledge_base_items:
        if item.item_type == "risk_keywords":
            risk_keywords.extend(item.keywords)
    
    risk_count = sum(1 for kw in risk_keywords if kw.lower() in transcription.lower())
    risk_score = max(0, 20 - risk_count * 5)
    score_components["risk_detection"] = risk_score
    
    # 4. 关键词命中（10分）
    all_keywords = []
    matched_keywords = []
    for item in knowledge_base_items:
        all_keywords.extend(item.keywords)
    
    for kw in all_keywords:
        if kw.lower() in transcription.lower():
            matched_keywords.append(kw)
    
    keyword_rate = len(matched_keywords) / max(len(all_keywords), 1)
    score_components["keyword_matching"] = keyword_rate * 10
    
    # 总分
    total_score = sum(score_components.values())
    
    return {
        "total_score": round(total_score, 2),
        "components": score_components,
        "covered_points": covered_points,
        "missing_points": [p for p in all_required_points if p not in covered_points],
        "risk_keywords_found": [kw for kw in risk_keywords if kw.lower() in transcription.lower()],
        "matched_keywords": matched_keywords
    }

def calculate_semantic_similarity(text1, text2):
    """计算两段文本的语义相似度"""
    from sentence_transformers import SentenceTransformer, util
    
    model = SentenceTransformer('all-MiniLM-L6-v2')
    embedding1 = model.encode(text1, convert_to_tensor=True)
    embedding2 = model.encode(text2, convert_to_tensor=True)
    
    cosine_score = util.cos_sim(embedding1, embedding2).item()
    return cosine_score
```

4.7.3 合规报告生成

```python
def generate_compliance_report(meeting_id, transcription, compliance_result):
    """生成合规分析报告"""
    report = {
        "meeting_id": meeting_id,
        "report_time": datetime.now().isoformat(),
        "overall_score": compliance_result["total_score"],
        "score_level": get_score_level(compliance_result["total_score"]),
        "detailed_scores": compliance_result["components"],
        "summary": {
            "covered_points_count": len(compliance_result["covered_points"]),
            "total_points_count": len(compliance_result["covered_points"]) + len(compliance_result["missing_points"]),
            "risk_items_count": len(compliance_result["risk_keywords_found"]),
            "matched_keywords_count": len(compliance_result["matched_keywords"])
        },
        "missing_points": compliance_result["missing_points"],
        "risk_keywords": compliance_result["risk_keywords_found"],
        "suggestions": generate_suggestions(compliance_result)
    }
    
    return report

def get_score_level(score):
    if score >= 90:
        return "优秀"
    elif score >= 75:
        return "良好"
    elif score >= 60:
        return "合格"
    else:
        return "不合格"

def generate_suggestions(compliance_result):
    suggestions = []
    
    if compliance_result["total_score"] < 60:
        suggestions.append("会议内容与政策要求差距较大，建议重新传达")
    
    if compliance_result["missing_points"]:
        suggestions.append(f"以下必传要点未覆盖：{', '.join(compliance_result['missing_points'])}")
    
    if compliance_result["risk_keywords_found"]:
        suggestions.append(f"发现风险内容：{', '.join(compliance_result['risk_keywords_found'])}，请核查")
    
    if not suggestions:
        suggestions.append("会议内容合规，继续保持")
    
    return suggestions
```
